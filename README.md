# Word-Counter

A simple, efficient, and user-friendly tool to count words, characters, and sentences in your text. 

## 🚀 Features

* **Word Count:** Accurately calculates the total number of words in a given text.
* **Character Count:** Counts total characters (with and without spaces).
* **Sentence Count:** Analyzes text to determine the number of sentences.
* **Fast and Lightweight:** Designed to process text quickly without unnecessary overhead.

## 🛠️ Technologies Used

* HTML, CSS, JavaScript
